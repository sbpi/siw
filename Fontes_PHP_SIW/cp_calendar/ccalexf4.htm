Calendar, Appel d'une fonction javascript

Retour à l'applet

Que montre cet exemple

Cet exemple monter un appel d'une fonction javascript à partir de l'applet. Notre fonction se nomme alertDate(). L'applet propose 3 fonctions qui peuvent être appelées d'une fonction javascript.
  • GetYear(), retourne un entier reprèsentant l'annéeacute;
  • GetMonth(), retourne un entier reprèsentant le mois
  • GetDay(), retourne un entier reprèsentant le jour

Example

Cliquez sur un jour pour lancer alertDate().

Code HTML

N'oubliez pas de rajouter le mot clé MAYSCRIPT dans le code HTML pour pouvoir appeler la fonction.
<applet
    code=ccalendar.class
    name=ccalendar MAYSCRIPT
    width=160
    height=160
	id = "cal">
<param name="color_fond" value="CCCCFF">
<param name="color_case" value="C0C0FF">
<param name="color_Comment" value="AAAAFF">
<param name="color_empty" value="9090FF">
<param name="jvf" value="alertDate()">
</applet>

Code javascript

function alertDate()
{
 alert("Year = "+document.ccalendar.getYear()+", month="+document.ccalendar.getMonth()+", day="+document.ccalendar.getDay());
}
