Votre choix

Cette exemple montre trois photos diffèrentes mais l'applet en accepte jusq'à 12 ( une par mois).

Code de l'applet

N'oubliez pas de rajouter le mot clé MAYSCRIPT dans le code HTML pour pouvoir mettre à jour les champs de la page.

<applet
    code=ccalendar.class
    name=ccalendar MAYSCRIPT
    width=210
    height=340
	id = "cal" >
<param name="lst_img" value="alison02.gif;alison03.gif;alison04.gif">
<param name="color_fond" value="FFCCCC">
<param name="color_case" value="FFC0C0">
<param name="color_Comment" value="FFAAAA">
<param name="color_empty" value="FF9090">
<param name="field" value="DateField">
<param name="form" value="DateForm">
</applet>
