Applet CCalendar, objet de type date

Cette applet peut être utilisée comme un objet de type date. Voici un exemple de forme avec un champs de type date que vous pouvez mettre à jour avec l'objet calendrier. Cliquez sur une des dates du calendier et le champs sera mis à jour avec votre choix. Le format de la date par défaut ( "d/m/a") a été conservé. Vous pouvez regarder la page en Anglais pour avoir un exemple avec une date de type "m.d.a".
Retour à l'applet

Que montre cet exemple

  • Mise à d'un champs de la page à partir de l'applet
  • Choix du format de la date ( voir page en anglais )

Code de l'applet

N'oubliez pas de rajouter le mot clé MAYSCRIPT dans le code HTML pour pouvoir mettre à jour les champs de la page.


<applet
    code=ccalendar.class
    name=ccalendar MAYSCRIPT
    width=160
    height=160
	id = "cal">
<param name="color_fond" value="CCCCFF">
<param name="color_case" value="C0C0FF">
<param name="color_Comment" value="AAAAFF">
<param name="color_empty" value="9090FF">
<param name="field" value="DateField">
<param name="form" value="DateForm">
</applet>
